Installation
============
The most recent release can be installed from
`PyPI <https://pypi.org/project/llm_adsorbate>`_ with:

.. code-block:: shell

    $ pip install llm_adsorbate

The most recent code and data can be installed directly from GitHub with:

.. code-block:: shell

    $ pip install git+https://github.com/schwallergroup/llm_adsorbate.git

To install in development mode, use the following:

.. code-block:: shell

    $ git clone git+https://github.com/schwallergroup/llm_adsorbate.git
    $ cd llm_adsorbate
    $ pip install -e .
